# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to business intelligence reporting. The project involves data loading, cleaning, exploratory data analysis (EDA), SQL-based data analysis, dashboard creation in Power BI, and presentation of insights through reports and slides.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

The dataset contains structured business data used for analysis and visualization.

Key dataset characteristics:

* Multiple attributes related to customer, sales, or business operations
* Structured tabular format
* Suitable for data cleaning, analysis, and reporting
* Imported and processed using Python

## Project Workflow

### 1. Data Loading

* Imported dataset into Python
* Verified structure and data types
* Performed initial inspection

### 2. Data Cleaning

* Removed duplicates
* Handled missing values
* Corrected inconsistent records
* Standardized data formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends
* Identified patterns and relationships
* Generated statistical summaries
* Created visualizations for deeper understanding

### 4. SQL Analysis

* Loaded cleaned data into relational databases
* Executed SQL queries for business insights
* Used filtering, aggregation, joins, grouping, and sorting operations
* Compared results across PostgreSQL, MySQL, and SQL Server environments

### 5. Dashboard Development

* Connected data to Power BI
* Built interactive dashboards
* Created KPIs and visual reports
* Designed user-friendly layouts for stakeholders

### 6. Reporting & Presentation

* Documented findings and recommendations
* Prepared project report
* Created presentation slides using Gamma AI and PowerPoint

## How to Run

### Clone Repository

```bash
git clone <repository-url>
cd Data-Analytics-Project
```

### Install Required Packages

```bash
pip install pandas numpy matplotlib seaborn
```

### Run Python Analysis

```bash
python analysis.py
```

### Execute SQL Queries

* Import data into PostgreSQL, MySQL, or SQL Server
* Run queries available in the SQL Queries folder

### Open Dashboard

* Open the Power BI (.pbix) file
* Refresh data connections if required
